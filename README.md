本分支从**myHomework**分支开出，对应实验三 - JSP+Servlet。

    项目结构：
    hotelbook - JavaWeb             //项目根目录
    |-- .idea                       //IDE配置文件
    |-- lib                         //第三方jar包
    |-- out                         //项目输出目录
    |-- src                         //代码
         |-- com.ink.hb

              |-- login             //登录实体类

         |-- layui                  //layui框架
         |-- WEB-INF
